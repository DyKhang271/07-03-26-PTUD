## Log (Quá trình thực hiện)

### Bước 1
Xây dựng phiên bản MVP của hệ thống quản lý sinh viên với các chức năng cơ bản:
- Thêm sinh viên
- Hiển thị danh sách sinh viên
- Sửa thông tin sinh viên
- Xóa sinh viên

### Bước 2
Thiết kế cơ sở dữ liệu SQLite để lưu trữ thông tin sinh viên.

### Bước 3
Xây dựng giao diện web để hiển thị danh sách sinh viên và form thêm/sửa dữ liệu.

### Bước 4
Mở rộng hệ thống theo yêu cầu:
- Thêm bảng Class
- Mỗi sinh viên thuộc một lớp

### Bước 5
Thêm các chức năng nâng cao:
- Tìm kiếm sinh viên theo tên
- Thống kê dữ liệu sinh viên
- Xuất dữ liệu ra file CSV
